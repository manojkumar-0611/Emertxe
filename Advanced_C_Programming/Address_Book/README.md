# Address Book

## Overview

Address Book is a console-based contact management application developed in C. It enables users to store, manage, search, edit, and delete contact information efficiently. The application demonstrates structured data management and persistent storage using file handling techniques.

## Features

* Add new contacts
* Search contacts by name or phone number
* Edit existing contact information
* Delete contacts
* Display all contacts
* Store data persistently using files

## Technologies Used

* C Programming
* Structures
* File Handling
* Dynamic Memory Management

## Build and Run

```bash
make
./address_book
```

## Learning Outcomes

* File handling operations
* Data organization using structures
* CRUD application development
* Modular programming techniques
