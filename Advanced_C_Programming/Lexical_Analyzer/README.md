# Lexical Analyzer

## Overview

Lexical Analyzer is a source code analysis tool that scans input programs and identifies lexical tokens such as keywords, identifiers, operators, literals, and symbols. It serves as the first phase of compiler design.

## Features

* Token identification
* Keyword detection
* Operator recognition
* Identifier classification
* Source code parsing
* Token reporting

## Technologies Used

* C Programming
* String Processing
* Compiler Design Concepts
* File Handling

## Build and Run

```bash
make
./lexer
```

## Learning Outcomes

* Compiler fundamentals
* Lexical analysis techniques
* Parsing concepts
* Text processing algorithms
