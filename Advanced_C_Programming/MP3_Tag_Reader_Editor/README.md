# MP3 Tag Reader & Editor

## Overview

This project provides functionality to read and modify metadata stored in MP3 files. It allows users to view and update information such as song title, artist, album, genre, and year.

## Features

* Read MP3 metadata tags
* Edit song information
* Display audio file details
* Support for ID3 tags
* Binary file manipulation

## Technologies Used

* C Programming
* File Handling
* Binary Data Processing
* Structures

## Build and Run

```bash
make
./mp3_editor
```

## Learning Outcomes

* Binary file processing
* Metadata management
* Structured data handling
* File format understanding
