# Advanced C Programming

This directory contains a collection of projects developed as part of the Advanced C Programming curriculum. These projects focus on applying core and advanced C programming concepts such as file handling, dynamic memory allocation, structures, pointers, command-line arguments, modular programming, and data processing.

## Objectives

* Strengthen problem-solving skills using the C programming language.
* Gain hands-on experience with real-world applications.
* Develop modular, maintainable, and efficient code.
* Understand file manipulation, memory management, and string processing techniques.
* Apply software development best practices in Linux environments.

## Projects

### Address Book

A console-based contact management system that allows users to add, search, edit, delete, and display contact information. The project demonstrates file handling, structures, and data management techniques.

### Steganography

A data-hiding application that embeds secret messages within image files and extracts them when required. This project showcases binary file operations and information security concepts.

### MP3 Tag Reader/Editor

A utility for reading and modifying metadata stored in MP3 files, including title, artist, album, genre, and year information. The project emphasizes binary file processing and structured data manipulation.

### Lexical Analyzer

A source code analysis tool that identifies and classifies tokens such as keywords, identifiers, operators, constants, and symbols. This project introduces compiler design fundamentals and text processing techniques.

### Source to HTML

A converter that transforms C source code into HTML format with syntax highlighting, making code easier to view and share through web browsers. The project demonstrates parsing, string manipulation, and file generation.
