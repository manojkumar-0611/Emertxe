# Source to HTML Converter

## Overview

Source to HTML Converter transforms C source code into a syntax-highlighted HTML document, making source code easier to read and share through web browsers.

## Features

* Convert C source code to HTML
* Syntax highlighting
* Preserve code formatting
* Generate browser-friendly output
* Automated file generation

## Technologies Used

* C Programming
* String Manipulation
* File Handling
* HTML Generation

## Build and Run

```bash
make
./source_to_html input.c output.html
```

## Learning Outcomes

* Parsing techniques
* File generation
* String processing
* Code visualization concepts
