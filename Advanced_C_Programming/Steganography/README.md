# Steganography

## Overview

Steganography is a data-hiding application that conceals secret information within image files without visibly altering the image. The project demonstrates information security concepts and binary file manipulation techniques.

## Features

* Hide secret text inside BMP images
* Extract hidden messages
* Password-protected encoding and decoding
* Binary file processing
* Data integrity verification

## Technologies Used

* C Programming
* File Handling
* Bit Manipulation
* Information Security Concepts

## Build and Run

```bash
make
./stego
```

## Learning Outcomes

* Binary file processing
* Bitwise operations
* Data encoding techniques
* Information hiding concepts
