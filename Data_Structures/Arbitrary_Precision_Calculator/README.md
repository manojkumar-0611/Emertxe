# Arbitrary Precision Calculator (APC)

## Overview

Arbitrary Precision Calculator performs arithmetic operations on numbers that exceed the limits of standard C data types. Large numbers are represented using linked lists for efficient processing.

## Features

* Large integer arithmetic
* Addition
* Subtraction
* Multiplication
* Dynamic number storage

## Technologies Used

* C Programming
* Linked Lists
* Dynamic Memory Allocation

## Learning Outcomes

* Big integer implementation
* Linked list manipulation
* Mathematical computation algorithms
