# Inverted Search

## Overview

Inverted Search creates an index of words from multiple text files and enables efficient keyword-based searching. The project demonstrates how search engines organize and retrieve information quickly.

## Features

* Create inverted indexes
* Search keywords efficiently
* Process multiple files
* Hash-based indexing
* Fast retrieval operations

## Technologies Used

* C Programming
* Hash Tables
* Linked Lists
* File Handling

## Learning Outcomes

* Search engine fundamentals
* Indexing techniques
* Hashing concepts
* Efficient data retrieval
