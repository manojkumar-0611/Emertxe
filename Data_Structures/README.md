# Data Structures Projects

This directory contains a collection of projects developed to strengthen my understanding of fundamental and advanced data structures, algorithms, and efficient data management techniques. These projects focus on implementing real-world applications using concepts such as linked lists, trees, stacks, queues, hashing, and dynamic memory management in C.

## Objectives

* Develop a strong foundation in data structures and algorithms.
* Understand efficient data organization and retrieval techniques.
* Apply abstract data structures to solve real-world problems.
* Improve problem-solving and analytical skills.
* Gain hands-on experience in implementing complex applications using C programming.

## Projects

### Inverted Search

An indexing and search application that creates an inverted index from multiple text files and enables fast keyword-based searching. The project demonstrates the use of hashing, linked lists, file processing, and efficient data retrieval techniques.

### Arbitrary Precision Calculator (APC)

A calculator capable of performing arithmetic operations on extremely large numbers that exceed the limits of standard data types. The project utilizes linked lists to represent and manipulate large integers efficiently.

### Red-Black Tree

An implementation of the self-balancing Red-Black Tree data structure supporting insertion, deletion, and searching operations while maintaining balance properties. The project demonstrates advanced tree algorithms and efficient data organization techniques.

### Text Editor

A console-based text editor that allows users to create, modify, insert, delete, and manage text dynamically. The project showcases the practical application of linked lists, dynamic memory allocation, and data manipulation.
