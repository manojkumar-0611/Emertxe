# Text Editor

## Overview

Text Editor is a console-based application that allows users to create, edit, and manage text dynamically. The project demonstrates practical use of linked lists and memory management techniques.

## Features

* Insert text
* Delete text
* Modify content
* Save and load files
* Dynamic text management

## Technologies Used

* C Programming
* Linked Lists
* Dynamic Memory Allocation
* File Handling

## Learning Outcomes

* Text processing techniques
* Dynamic memory management
* Linked list applications
* User-driven application design
