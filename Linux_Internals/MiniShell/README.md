# MiniShell

## Overview

MiniShell is a custom Linux shell developed in C that mimics the behavior of a standard Unix shell. The shell interprets user commands, creates child processes, executes programs, and manages foreground and background jobs.

This project provides practical experience with Linux process management, signals, pipes, and command execution mechanisms.

## Features

* Execute Linux commands
* Foreground and background process execution
* Built-in commands
* Signal handling
* Command history support
* Pipe implementation
* Input and output redirection
* Process management

## Technologies Used

* C Programming
* Linux System Calls
* Process Management
* Signals
* Pipes
* GCC Compiler

## Build and Run

```bash
make
./minishell
```

## Learning Outcomes

* Understanding shell architecture
* Process creation using fork()
* Program execution using exec family functions
* Signal handling and job control
* Inter-process communication
