# Multi-Threaded Inverted File Index

## Overview

This project implements an efficient inverted indexing system using multithreading. Multiple files are processed concurrently to create an index that enables fast keyword-based searching across large datasets.

The project combines data structures, file handling, and thread synchronization techniques to improve performance.

## Features

* Multi-threaded file processing
* Inverted index generation
* Fast keyword search
* Efficient memory utilization
* Thread synchronization
* Large dataset support

## Technologies Used

* C Programming
* POSIX Threads (Pthreads)
* Data Structures
* File Handling
* Synchronization Mechanisms

## Build and Run

```bash
make
./inverted_index
```

## Learning Outcomes

* Multithreaded programming
* Thread synchronization
* Concurrent data processing
* Performance optimization
* Search engine indexing concepts
