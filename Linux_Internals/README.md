# Linux Internals Projects

This directory contains projects developed as part of Linux Internals and System Programming training. These projects focus on process management, inter-process communication, networking, multithreading, socket programming, file systems, and Linux system calls.

## Objectives

* Understand Linux operating system internals.
* Gain hands-on experience with system-level programming.
* Develop networking applications using TCP/IP protocols.
* Learn process creation, synchronization, and communication.
* Implement multithreaded applications for improved performance.
* Apply Linux APIs and system calls to solve real-world problems.

## Projects

### MiniShell

A custom Linux shell that supports command execution, process management, I/O redirection, pipes, background execution, and built-in commands. The project provides practical exposure to process creation, signals, and command-line interpretation.

### TFTP Server and Client

An implementation of the Trivial File Transfer Protocol (TFTP) using socket programming. The project enables file transfer between a client and server over a network while demonstrating UDP communication and protocol handling.

### TCP/IP Chat Room

A multi-client chat application developed using TCP sockets. The project allows multiple users to communicate simultaneously through a centralized server, showcasing network programming and client-server architecture.

### Remote Diagnostics

A client-server application that allows remote monitoring and diagnostics of system resources. The project demonstrates socket programming, system command execution, and remote data retrieval in Linux environments.

### Multi-Threaded Inverted File Index

A high-performance indexing application that uses multiple threads to process and index text files efficiently. The project combines multithreading, synchronization mechanisms, file handling, and data structures to improve search performance.
