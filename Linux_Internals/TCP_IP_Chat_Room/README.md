# TCP/IP Chat Room

## Overview

TCP/IP Chat Room is a multi-client communication application developed using TCP sockets. The application allows multiple users to connect to a server and exchange messages in real time.

The project demonstrates client-server architecture and concurrent communication using networking concepts.

## Features

* Multiple client support
* Real-time messaging
* TCP socket communication
* Client connection management
* Message broadcasting
* User session handling

## Technologies Used

* C Programming
* TCP/IP Networking
* Socket Programming
* Linux System Programming

## Build and Run

### Server

```bash
make
./server
```

### Client

```bash
./client <server_ip> <port>
```
