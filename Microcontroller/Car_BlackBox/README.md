# Car Black Box

## Overview

Car Black Box is an embedded system project designed to record and monitor vehicle events. Similar to an aircraft black box, the system continuously logs important operational data and stores it in non-volatile memory for future analysis.

The project provides a practical understanding of embedded systems, peripheral interfacing, and real-time event logging.

## Features

* Real-time event monitoring
* Speed and gear tracking
* Event logging with timestamps
* EEPROM-based data storage
* LCD display interface
* Menu-driven user interaction
* UART-based data download

## Hardware Components

* PIC Microcontroller
* LCD Display
* Matrix Keypad
* RTC Module
* EEPROM
* UART Interface

## Technologies Used

* Embedded C
* MPLAB IDE
* XC8 Compiler
* EEPROM Interface
* RTC Interface
* UART Communication

## Learning Outcomes

* Embedded firmware development
* Peripheral interfacing
* Real-time clock management
* Non-volatile memory handling
* Menu-driven embedded applications
