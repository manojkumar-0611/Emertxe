# Microcontroller Projects

This directory contains embedded systems projects developed using microcontrollers and peripheral interfacing techniques. These projects focus on real-time system design, hardware-software integration, sensor interfacing, and embedded programming concepts.

## Projects

### Car Black Box

A microcontroller-based event data recorder designed to monitor and log critical vehicle events. The system records parameters such as speed, gear position, and accident events, providing valuable information for diagnostics and analysis.

## Technologies Used

* Embedded C
* Microcontrollers
* EEPROM
* RTC (Real Time Clock)
* LCD Display
* Keypad Interface
* UART Communication

## Learning Outcomes

* Embedded system design
* Peripheral interfacing
* Real-time data logging
* EEPROM memory management
* Microcontroller programming
