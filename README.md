# EMERTXE Projects Portfolio

## Overview

This repository contains projects completed as part of my training at EMERTXE Information Technologies. The projects cover a wide range of domains including Data Structures, Advanced C Programming, Linux Internals, System Programming, Networking, Multithreading, and Embedded Systems.

The objective of these projects was to strengthen problem-solving skills, gain hands-on experience with real-world applications, and develop a strong foundation in software and embedded systems development.
